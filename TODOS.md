## D3モジュラーインポートの最適化
- **What:** D3をフルバンドル（`import * as d3`）ではなく、使用する機能のみ個別インポート（`import { select } from 'd3-selection'`）に変更する
- **Why:** フルバンドル（~500KB）から使用モジュールのみ（~100-150KB推定）に削減でき、初期ロード時間が改善される。静的サイトなのでビルドサイズがそのままユーザー体験に直結する
- **Depends on:** チャートコンポーネントの実装完了後（どのD3モジュールを使うか確定してから）
